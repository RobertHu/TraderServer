Menu

The Menu contains buttons that allow user to activate or to perform the following function quickly:
  • Instrument - when the button is pressed, the item selection box will appear that enables investor to select the trading instruments to view. See Instrument
  • Passwords - Investor may change the logon passwords any time at wish. See change passwords
  • Chart - the investor may view the interday/ intraday charts by pressing the button. The intraday chart shows the daily movement of the trading instrument. See Charts
  • Message - the system provides communication from the trading desk to the user through Message. See Message
  • Report - the button issues command to request from the server for the account statement or ledger depending on the selection of the investor. See Reports
