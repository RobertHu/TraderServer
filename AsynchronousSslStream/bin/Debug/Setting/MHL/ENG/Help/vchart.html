Charts

 

The Chart dialog is activated through mouse click at the Chart button in the Menu of the main screen. The dialog is mainly consist of:

  • Selection Bar
  • Chart Area

    • Chart dialog

    Selection Bar

    On the top of the chart window, a buttons bar is provided to allow user to set various chart properties and functions.

    Buttons Bar:

    • Type  - the following chart types are available for user selection:
    • Real Time
    • 1 minute
    • 5 minute
    • Daily
    • Weekly
    • Monthly
      •
    • Item  - a pull down box provides user to select charting instrument
    • Style Type  - the following are available for user sel ection:
    • Candle
    • High HiLo
    • Close HiLo
    • Open CloseHiLo
      •
    • A & I  -for selection of Analysis & indicator

      • Analysis selection

       

      Indicator selection

       
    • Clear  -  to clear all lines drawn by the user
    • Draw  -  to enable the user to draw lines on the graph
