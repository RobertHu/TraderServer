미확인된 지시사항 취소하기

회사의 거래 규칙에 따라, 미확인 상황에 있을 때 사용자의 제한/정지 (Limit/Stop), MOO나 MOC 주문 취소가 가능하다. 

미확인 주문 취소를 위해서는 다음과 같이 하세요:
  • 실행 주문 패널 상의 주문을 마우스로 클릭한다. 

  • 거래 지시사항 대화창이 활성화되면, 취소 버튼을 클릭한다.

  • 취소 요청의뢰를 확인하는 대화창이 나타나면 OK 버튼을 클릭한다.

  • 거래 지시사항 대화창을 나가서 주 스크린으로 돌아 간다. 
    •
     

    요청을 의뢰하면, 실행 패널 상의 주문 상태가 미확인에서 요청이 수락되면 취소로 바뀔 것이다.
