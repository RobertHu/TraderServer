Operating system

The system is designed to be run on Microsoft Windows 98SE, Microsoft Windows Me, Microsoft Windows 2000 and Microsoft Windows XP. If you are using Windows 2000 or lower version, we strongly suggest you upgrade the Internet Explorer to version 6.0 or higher.

If you are running Internet Explorer 5.0 on Microsoft Windows 98/Me/NT, you may need to install the Windows Script 5.6 before the system could run properly.

It is also recommended that you run the system with a display resolution of 1024 x 768.

If you find any problem in running the system on your computer, kindly contact our technical support.
