帐户状态

该面板跟踪所有的帐户交易活动，允许用户实时地监视交易帐户的形势。

如果为单独一个交易帐户则会在主窗口左下角直接显示其信息。

如有多个交易帐户则帐户状态面板将把其全部以列表形式显示出来，当用户想查看某一帐户状态时可以双击相应的帐户打开查看窗口：
多交易帐户状态

帐户状态窗口为用户显示下列信息：

  • Code  - 交易帐户
  • Curr.  - 交易帐户的基础货币，如为多货币帐户则在输出报表时将以此货币作为结算货币。
  • Balance  - 帐户结余，其货币单位为交易帐户的基础货币。
  • Necessary  - 帐户交易指令的必须保证金数量，其货币单位为交易帐户的基础货币。.
  • Float P/L  - 指令实时浮动盈亏，其货币单位为交易帐户的基础货币。.
  • Equity  - 交易帐户的实时有效保证金，其货币单位为交易帐户的基础货币。
  • Memo  - 帐户出现某种危急信状况时的用户提示信息。
